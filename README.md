# Анализ веб-приложения Demoblaze

## Описание проекта
В рамках данного проекта проводится анализ веб-приложения Demoblaze — интернет-магазина электроники.  
Сайт предоставляет пользователям возможность просматривать каталог товаров, изучать информацию о продуктах, добавлять товары в корзину и оформлять заказ через веб-интерфейс.

Целью работы является изучение структуры и функциональности существующего веб-приложения и подготовка технического задания на основе анализа его работы.

Ссылка на сайт: https://www.demoblaze.com

### Инструкция по запуску
1. Установить Python.
2. Открыть терминал в папке проекта.
3. Создать виртуальное окружение:
```sh
python -m venv .venv
```
4. Активировать виртуальное окружение:
```sh
.venv\Scripts\activate.ps1
```
Разрешить выполнение скриптов:
   `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

5. Установить зависимости:
```sh
pip install -r requirements.txt
```
6. Запустить все тесты:
```sh
pytest
```
